# Game-Off-2018

Github Game Off 2018 - Themed HYBRID!

## Setup

### 1. Clone this repo:

In your workspace directory.

Clone:

```git clone ```  

### 2. Install Node.js and npm:

https://nodejs.org/en/

#### 2.1 Npm optional: Install yarn

https://yarnpkg.com/

### 3. Install dependencies

Into cloned repo's directory.

Run with npm:
```npm install```
or run with yarn:
```yarn install```
### 4. Development server:
Run:
```npm run start```
or
```yarn run start```
